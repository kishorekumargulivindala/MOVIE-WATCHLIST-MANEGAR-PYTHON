# movie-watchlist-manager-python
--------Movie Watchlist Manager-------

A simple Python project to manage your movie watchlist.

Description:

This project allows you to add, view, edit, delete, and track movies you want to watch.
You can also mark movies as watched and give ratings.

Features:

- 1.Add movies
- 2.View movie list
- 3.Delete movies
- 4.Mark as watched
- 5.Rate movies (1–5)
- 6.Edit movie names

Technologies Used:

- Python
- JSON (for storing data)

How to Run:

1. Install Python
2. Download or clone this repository
3. Run the file:

python movie_watchlist.py

Files:

- "movie_watchlist.py" → Main program
- "movies.json" → Stores movie data

Project Idea:

This is a beginner-friendly project built to practice Python concepts like:

- Functions
- Loops
- File handling
- JSON data
